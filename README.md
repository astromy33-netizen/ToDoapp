# TodoList

Простое веб-приложение для управления списком задач.

## Функциональность

- Добавление новых задач
- Отметка задач как выполненные (кнопка с галочкой)
- Удаление задач
- Сохранение состояния задач (пока страница не перезагружена)

## Технологии

- HTML5
- CSS3
- JavaScript (без использования фреймворков)


## Структура проекта

- `index.html` - основная HTML-разметка
- `style.css` - стили оформления
- `script.js` - логика приложения
