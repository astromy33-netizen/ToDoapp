# TodoList

Простое веб-приложение для управления списком задач.

## Функциональность

- Добавление новых задач
- Отметка задач как выполненные (кнопка с галочкой)
- Удаление задач
- Сохранение состояния задач (пока страница не перезагружена)

## Технологии

- HTML5
- CSS3
- JavaScript (без использования фреймворков)

## Как использовать

1. Откройте файл `index.html` в браузере
2. Введите задачу в поле ввода
3. Нажмите кнопку "Добавить" или клавишу Enter
4. Используйте кнопку с галочкой (✓) для отметки выполненных задач
5. Используйте кнопку "Удалить" для удаления задач

## Структура проекта

- `index.html` - основная HTML-разметка
- `style.css` - стили оформления
- `script.js` - логика приложения
